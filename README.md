## ✅ 已实现功能

### 🚀 性能

- **启动速度** — 冷启动快，打开即用
- **内存占用小** — 多标签浏览时内存占用更低
- **网页性能** — [Speedometer](https://browserbench.org/Speedometer3.0/) 跑分表现领先（具体分值因系统与硬件而异，仅供参考）

### 🔄 账号与数据同步

- **双账号登录** — Turbo / Google；头像菜单、登录页、设置 > 个人资料 / 同步与历史页均可切换
- **数据同步** — Turbo 账号走 Turbo 云同步，Google 账号走 Google 同步；支持书签、浏览历史、密码、Passkey、浏览器设置、扩展程序

### 📦 数据迁移与导入

- **自 Chrome / Edge 迁入** — 书签、密码、自动填充、扩展（以版本支持为准）

### 🛡️ 应用商店与扩展

- **Chrome / Edge 双商店** — 菜单、扩展程序管理页（`chrome://extensions`）均可进入
- **Crx搜搜** — 扩展管理页侧栏提供入口；支持从 Crx搜搜等可信来源一键下载并安装 `.crx`
- **Edge 主题商店** — 标题栏与侧边栏主题入口
- **AdGuard 广告拦截** — 过滤网页广告、跟踪器与弹窗，页面更干净
- **沉浸式翻译** — 双语对照网页翻译，支持划词翻译与视频字幕等

### 🖱️ 鼠标手势与老板键

- **老板键** — 默认 `Ctrl+Q` 隐藏/显示窗口，可自定义快捷键
- **鼠标手势** — 按住右键拖动，路径可视化；默认 ↑↓ 滚动、← 后退、→ 前进，可自定义
- **超级拖拽** — 将链接、文本、图片拖到页面空白处，在后台标签页打开（可在手势设置中开关）

### ✂️ 界面与效率

| 能力         | 说明                                                                    |
| ------------ | ----------------------------------------------------------------------- |
| **系统主题** | 跟随系统浅色 / 深色                                                     |
| **一键换肤** | 标题栏一键切换浏览器外观配色                                            |
| **分屏对照** | 工具栏分屏按钮，左右两栏并排浏览                                        |
| **截图**     | 工具栏左键全屏、右键网页导出（图片 / PDF）；`Ctrl/Cmd+Shift+X` 区域截图 |
| **网络代理** | 可选系统代理；关闭后可配置自定义代理                                    |
| **多线程下载** | 设置 → 下载中可开关「开启多线程下载加速」（默认开启）                   |
| **沉浸模式** | 自动隐藏标签栏与工具栏，鼠标移到窗口顶部再显示；`Ctrl/Cmd+F11`；macOS「显示」菜单可开关 |
| **新标签页** | 内置新标签页，支持壁纸、搜索增强与全页定制（图标、布局等）              |
| **工具栏**   | 恢复网页、截图入口与定制                                                |

### 🏷️ 标签与地址栏

- **垂直标签** — 内置垂直标签能力，默认横向布局；可通过标签栏「标签布局切换」按钮，或设置 > 外观 > 标签页位置切换横/竖布局
- **标签** — 双击关闭；阻止关闭最后标签；工具栏「恢复网页」左键恢复最后标签，长按或右键查看完整列表
- **粘贴即打开** — 右键标签栏「新建标签页」按钮，读取剪贴板：链接直接打开，文字按默认搜索引擎搜索
- **地址栏** — 悬停显示二维码；一键复制链接

### ⬆️ 升级与安装

- **内核** — 持续跟进 Chromium 主线；具体 Chromium 版本见 [下载安装](https://tbrowser.cn/zh/guide/download.html)
- **更新** — 设置页检查更新，后台自动检查
- **安装** — 首次运行引导；Windows 桌面快捷方式与固定任务栏
- **跨平台** — Windows / macOS
